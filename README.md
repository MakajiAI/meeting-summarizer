# Meeting Summarizer

Gemini APIを使用した、ローカル環境で動作する議事録要約AIツールです。

テキスト形式の会議議事録を読み込み、会議の目的・決定事項・次のアクション・全体要約をMarkdown形式で出力します。
CLI版とGUI版の両方に対応しており、プログラミングに慣れていない人でもGUIからファイルを選択して要約を作成できます。

---

## 概要

このツールは、UTF-8形式のテキストファイルを読み込み、Google Gemini APIに送信して議事録要約を生成します。

要約結果は画面に表示されるだけでなく、入力ファイルと同じフォルダ、または指定した保存先に `.txt` ファイルとして保存されます。

出力される要約には、以下の内容が含まれます。

* 会議の目的
* 決定事項
* 次のアクション
* 全体要約
* 入力ファイル名
* 処理日時
* 使用モデル名

### 2. 仮想環境を作成

```powershell
python -m venv .venv
```

もし上記でエラーが出る場合は、以下を試してください。

```powershell
.\.venv\Scripts\activate.bat
```

```powershell

```

## .env の設定方法

このツールを使用するには、Gemini APIキーが必要です。

まず、`.env.example` をコピーして `.env` を作成します。

```powershell
copy .env.example .env
```

作成した `.env` を開きます。

```powershell
notepad .env
```

中身を以下のように編集してください。

```env
GEMINI_API_KEY=your_api_key_here
```

`your_api_key_here` の部分を、自分のGemini APIキーに置き換えてください。

`.env` にはAPIキーが含まれるため、GitHubには公開しないでください。
このリポジトリでは `.gitignore` により `.env` をGit管理から除外しています。

## CLI版の使い方

### 基本実行

```powershell
python main.py sample-meeting.txt
```

実行すると、議事録要約がターミナルに表示され、同じフォルダに以下のようなファイルが保存されます。

```text
summary_sample-meeting.txt
```

### 保存先を指定する

`--output` オプションを使うと、保存ファイル名を指定できます。

```powershell
python main.py sample-meeting.txt --output result.txt
```

この場合、要約結果は `result.txt` に保存されます。

---

## GUI版の使い方

GUI版は以下のコマンドで起動できます。

```powershell
python gui.py
```

GUIでは以下の操作ができます。

1. 「選択」ボタンで議事録テキストファイルを選択
2. 保存先ファイル名を指定
3. 「要約開始」ボタンをクリック
4. 要約結果を画面で確認
5. 要約結果が指定したファイルに保存される

CLIに慣れていないユーザーでも、ファイル選択とボタン操作だけで議事録要約を作成できます。

```text
鈴木：私が担当します。6月30日までに告知素材を作成します。

次回：6月30日 14:00
```

```markdown
* 鈴木が2026年6月30日までにSNS告知素材を作成する。
```

## エラー時の確認方法

### 入力ファイルが存在しない場合

```powershell
python main.py no-file.txt
```

表示例：

```text
エラー: 入力ファイルが見つかりません: no-file.txt
```

### 入力ファイルが空の場合

空の `empty.txt` を作成して実行します。

```powershell
New-Item empty.txt -ItemType File
python main.py empty.txt
```

表示例：

```text
エラー: 入力ファイルが空です: empty.txt
```

### APIキーが設定されていない場合

`.env` が存在しない、または `GEMINI_API_KEY` が設定されていない場合、APIキーに関するエラーが表示されます。

`.env` の中身を確認してください。

```env
GEMINI_API_KEY=your_api_key_here
```

### 保存したファイルが文字化けして見える場合

PowerShellの `type` コマンドで日本語が文字化けする場合があります。
その場合は、以下のようにUTF-8を指定して確認してください。

```powershell
Get-Content summary_sample-meeting.txt -Encoding UTF8
```

## 注意事項

実際の会議議事録には、機密情報や個人情報が含まれる可能性があります。
このツールはGemini APIにテキストを送信して要約を作成するため、APIに送信してよい内容か確認してから使用してください。

社外秘情報、個人情報、契約情報、未公開情報などを含む議事録を扱う場合は、利用前に十分注意してください。
